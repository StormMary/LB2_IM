# LB2_IM
Проект реализует имитационную модель деятельности магазина с учётом:
кредитования магазина при недостатке средств;
налогового учёта и периодических выплат каждые 30 дней;
Модель основана на исходной схеме, построенной в Powersim Studio, и преобразована в код на языке C++, чтобы продемонстрировать алгоритмическую реализацию системной динамики.
